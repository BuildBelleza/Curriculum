# Server 101

## Simple Server with Express

## We use the `express` library to build servers.

### Steps to build a simple server

Follow these steps in the project root folder.

- `npm init -y`
- `npm i express`
- `npm i -D nodemon`
- `toucher server.js`
- add the code to build a server (see **demo/**)
- `npx nodemon ser.js`
